# Release Pack v1.0.0 — rldyourterm

## Предназначение
Единый ручной чеклист для выпускной сборки без CI.

## Требования перед релизом
1. Версия и документация:
   - `1.0.0` зафиксирован в `AGENTS.md`, `planning/v1.0.0-development-blueprint.md`, `metrics/version/1.0.0/*`.
   - все ADR/риск/рули закрыты.
2. Тех-гейты:
   - `planning/quality/v1.0.0-acceptance-matrix.md` выполнен по всем секциям.
   - `planning/quality/v1.0.0-quality-gates.md` в состоянии Alpha/Beta/RC.
3. Тестовый пакет:
   - `planning/operations/v1.0.0-manual-test-plan.md` пройден на всех целевых OS.
4. Риск-контроль:
   - все high/medium риски имеют mitigation и owner.

## Релизная последовательность (ручная)
1. Сформировать release build.
2. Проверить startup на всех целевых ОС.
3. Выполнить `manual-test-plan` с записью результатов.
4. Сформировать SHA checksum артефактов.
5. Подготовить release notes:
   - what changed,
   - blockers fixed,
   - GPU/CPU behavior notes.
6. Подготовить GitHub release (manual).
7. Подготовить scripts для install path:
   - GitHub installer bootstrap,
   - roadmap note for apt/homebrew packaging.
8. Финальный чек: все документы и метрики обновлены и зафиксированы.
